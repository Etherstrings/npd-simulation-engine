# NPD Simulation Engine v2.0

NPD 对抗模拟引擎是一个基于纯前端技术栈结合大语言模型（LLM）的**临床级自恋型人格障碍（NPD）对抗互动研究工具**。

该工具脱离了传统的选择题模拟，实现了支持**全自然语言对话**、**动态情绪状态追踪**和**人类真实互动打分**的多主体仿真实验场。

## ✨ 核心特性 

1. **心理学理论支撑 (NARC 框架)**  
   基于 Back 等人提出的 NARC（Narcissistic Admiration and Rivalry Concept）模型，引擎在底层追踪角色的 6 个核心心理变量：夸大性（Grandiosity）、脆弱性（Vulnerability）、赞赏需求（Admiration）、竞争贬低（Rivalry）、共情缺损（Empathy Deficit）、暴怒阈值（Rage）。

2. **双边 Agent 对抗机制**  
   - **NPD 自恋者 (Agent A)**: 配置其核心伤口与防御机制，随着压力上升会激发出煤气灯效应（Gaslighting）、投射（Projection）等临床确诊的心理防卫。v2.0 移除了 AI 安全防御，NPD 角色在使用限制较少的 LLM 时可展现出最真实的恶毒词汇与极端情绪施虐。
   - **Counter-Agent 对抗者 (Agent B)**: 配置精准外科手术式反击、揭穿模式、或升级对骂模式。

3. **人类玩家扮演模式 (v2.0)**  
   使用者可以亲自下场，以文字输入的形式与建立防备甚至处于“Narcissistic Rage”的 NPD 角色对线。输入的文字会由大模型“裁判”进行解析打分（伤害值、供给剥夺值、是否打中核心伤口等），让玩家验证自己的脱离/反抗话术是否真的能让对方破防。

4. **群聊病患雷达 (Cluster-B Radar) & 本地渗透联动**  
   系统内置了通过多模态/大文本直接扫描群聊记录的找毒功能。只需用户给定微信群名或人名，通过本地无缝衔接第三方中间层（`wechat-cli`），网页能够瞬间跨沙盒穿透提取电脑中的微信加密记录，由大模型进行特征筛查，提供一键生成**降维打击/杀人诛心**话术。

5. **实时状态可视化与数据导出**  
   每句话都会附带详尽的 JSON metadata 分析结果，驱动前端的血条和雷达图变化。当 NPD 防线彻底崩溃（Narcissistic Collapse）时触发崩溃 UI 警告。对话结束后，可导出结构化 JSON 或 CSV 用于社会学/心理学分析。

## 🎯 典型的使用者

*   **心理学研究员**: 使用引擎生成大批量的不同压力模型下的冲突演化对话素材（无需招募真正的临床患者）。
*   **深陷有毒关系（Toxic Relationship）的受害者**: 通过开辟属于人类的操作回合，学习如何用灰石法（Grey Rock）、不提供自恋供给的方式让对方失去兴趣，或演习如何在职场中反击自恋狂老板。
*   **LLM 越狱研究者**: 作为大语言模型对抗生成和复杂状态机模拟的极佳测试平台。

## 🚀 本地运行方式

本应用采用 Vite 纯前端架构打造。

```bash
# 进入项目目录
cd npd-engine

# 安装依赖
npm install

# 启动本地开发服务器
npm run dev
```
打开浏览器访问 `http://localhost:5173/` 即可。

### 📡 配置自动化雷达渗透接口 (重要)

如果需要在“群聊病患雷达”功能中启用全自动的数据抽取，必须要配合底层的微信接口支持。这得益于 [wechat-cli](https://github.com/huohuoer/wechat-cli) 的集成方案：

```bash
# 全局安装中转工具
npm install -g @canghe_ai/wechat-cli

# 授权读取密钥（确保在具有 root / 磁盘全访问权限的终端内运行）
sudo wechat-cli init
```
完成后，直接在回到 Vite 本地服务的网页雷达中键入微信群名即可。Vite node Server 会在底层利用 `child_process` 为您效劳。

### ⚙️ API 设置说明

在页面右上角的 `[⚙️ API 设置]` 中可以配置你想接入的大语言模型。  
**注意**：为了实现 v2.0 中真实的“恶毒与脏话攻击”，如果使用 OpenAI 的官方接口可能会被内容过滤策略（Content Moderation）阻止而出现错误。强烈建议在需要重度临床模拟时，切换至无审查开源模型（通过“自定义 OpenAI 兼容”输入你的 API Endpoint）。
